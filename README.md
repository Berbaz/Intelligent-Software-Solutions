
# AI in Software Engineering — Assignment Project

This repository contains all deliverables for the assignment themed **"Building Intelligent Software Solutions"**.

## Contents
- **breast_cancer_wisconsin.csv** — Dataset for Task 3.
- **report.pdf** — Theoretical analysis, case study, practical results, and ethical reflection.
- **docugenie.md** — Innovation Challenge proposal.
- **Code folder** — Sorting scripts, Selenium tests, predictive analytics notebook.

## How to Run
1. Install Python 3.8+  
2. Install required libraries:
   ```
   pip install -r requirements.txt
   ```
3. Run Jupyter Notebook for Task 3:
   ```
   jupyter notebook notebooks/predict_priority.ipynb
   ```
4. Run Selenium tests:
   ```
   pytest tests/test_login.py
   ```
